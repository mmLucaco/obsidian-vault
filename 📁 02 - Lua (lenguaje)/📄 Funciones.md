
## 🧩 Concepto simple

> Bloques de código reutilizables.

---

## ⚙️ Concepto técnico

> Subrutinas que encapsulan lógica y pueden recibir parámetros.

---

## 🏗️ Cuándo se usa

- Reutilizar código
    
- Separar lógica
    

---

## 🧪 Ejemplo

```lua
function sumar(a, b)
    return a + b
end
```

---

## 🎮 Ejemplo en Roblox

```lua
function darMonedas(player)
    player.coins += 10
end
```
