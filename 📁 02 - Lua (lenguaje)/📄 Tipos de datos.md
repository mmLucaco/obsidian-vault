
## 🧩 Concepto simple

> Son los diferentes tipos de valores que pueden tener las variables.

---

## ⚙️ Concepto técnico

> Clasificaciones de valores que determinan cómo se almacenan y manipulan en memoria.

---

## 🏗️ Cuándo se usa

- Siempre que creás una variable
    
- Para validar datos
    
- Para evitar errores en lógica
    

---

## 📚 Tipos en Luau

- [[nil]]
    
- [[boolean]]
    
- [[number]]
    
- [[string]]
    
- [[Table]]
    

---

## 🎮 Ejemplo en Roblox

```lua
local coins = 100 -- number
local name = "Lucas" -- string
local isAlive = true -- boolean
```
