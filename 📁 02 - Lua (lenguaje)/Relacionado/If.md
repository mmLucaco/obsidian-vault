
## 🧩 Concepto simple

> Ejecuta código si se cumple una condición.

---

## ⚙️ Concepto técnico

> Estructura de control que evalúa expresiones booleanas.

---

## 🏗️ Cuándo se usa

- Validaciones
    
- Lógica del juego
    

---

## 🧪 Ejemplo

```lua
if x > 10 then
    print("Mayor")
end
```

---

## 🎮 Ejemplo en Roblox

```lua
if playerCoins >= 100 then
    print("Puede comprar")
end
```
