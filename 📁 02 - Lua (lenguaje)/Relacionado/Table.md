

## 🧩 Concepto simple

> Lista o diccionario.

---

## ⚙️ Concepto técnico

> Estructura de datos clave-valor dinámica.

---

## 🏗️ Cuándo se usa

- Guardar datos complejos
    
- Sistemas de inventario
    
- Datos de jugador
    

---

## 🧪 Ejemplo

```lua
local player = {
    name = "Lucas",
    coins = 100
}
```

---

## 🎮 Ejemplo en Roblox

```lua
player.coins = player.coins + 10
```
